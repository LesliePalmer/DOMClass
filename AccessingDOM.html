Accessing Elements

Using the ID (#header)

Access this by using getElementById() returns an element node

Using the Class (.header)

1. Access this by using getElementByClassName() returns a live HTMLCollection
2. Access this by using getElementByClassName()

Using the Tag (paragraph tag)

1. Access this by using getElementByTagName() returns an element node

2. Access this by using getElementByTagName()

Using the Query Selector (with ID of 'query')

1. Access this by using querySelector() returns an element node

Using the Query Selector All (with class of 'query-all')

1. Access this by using querySelectorAll()
2. Access this by using querySelectorAll()
